О системе

Целью сайта является предоставление рекомендаций по увеличению прибыли на установленное значение путём решения обратной задачи формирования прибыли.

На этом сайте Вы можете:

  • Увидеть оформленную информацию по продажам
  • Получить рекомендации по цене, количеству и себестоимости продаваемого товара на основе решения обратной задачи
Вы можете воспользоваться этими возможностями уже сейчас, загрузив файл с отчётом о продажах в форме ниже.

После загрузки файла Вы увидите страницу с данными, полученными из него.
